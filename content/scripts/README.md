# Scripts

Verbatim language the team uses on live calls. If a script sentence is here, it's the one QA scores against.

Categories:
- `csm/` — inbound call handling (greeting, empathy, intake, booking confirmation, closing).
- `sales/` — in-home diagnostic → quote → close scripts.
- `objections/` — one file per common objection with the recommended response.

Scripts reference pricing via `{{price:token}}` tokens. Never restate a price inline.
